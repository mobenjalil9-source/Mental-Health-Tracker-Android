# Mental Health Tracker App

## Description
An Android application developed in Java to track daily mood and stress levels using SQLite.

## Features
- Mood tracking
- Stress level monitoring
- Offline data storage
- Simple UI

## Technologies Used
- Android Studio
- Java
- SQLite

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or device
